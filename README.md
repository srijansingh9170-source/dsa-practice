Data Structures and Algorithms Practice

Description
This repository contains solutions to various Data Structures and Algorithms problems implemented in C++. The goal is to improve problem solving skills and strengthen understanding of fundamental programming concepts.

Topics Covered

- Arrays
- Strings
- Recursion
- Sorting Algorithms
- Searching Algorithms

Features

- Clean and well-structured code
- Beginner friendly solutions
- Useful for coding interview preparation

Installation

1. Clone the repository

git clone https://github.com/srijansingh9170-source/dsa-practice.git

2. Open the files in your preferred C++ IDE or compiler.

Usage

Compile and run any file.

Example using g++:

g++ reverse_number.cpp  
./a.out
